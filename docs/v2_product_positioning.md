# FinCopilot v2 产品定位文档

## 1. v2 定位

FinCopilot v2 是一个面向小微企业的 Agentic CFO Copilot Demo，帮助用户围绕现金流安全、发票压力、异常支出和财务目标，完成数据整理、风险识别、行动计划生成和进展跟踪。

## 2. 目标用户

主要用户包括：

- 小微企业主；
- 小型工作室负责人；
- 自由职业团队负责人；
- 早期创业团队运营者。

这些用户通常没有专职财务团队，需要用低成本方式完成基础财务运营管理。

## 3. 核心现实问题

### 3.1 现金流风险不透明

小微企业的收入和支出经常错位。客户回款可能延迟，但供应商付款、房租、工资和软件订阅仍需按时支付。

### 3.2 发票和付款任务分散

用户需要同时跟踪未支付发票、逾期发票、即将到期发票和供应商付款，容易遗漏关键任务。

### 3.3 异常支出难以及时处理

云服务费用突增、供应商付款异常、重复扣费和罕见商户大额支出都可能影响经营现金流。

### 3.4 财务目标缺少执行路径

用户可能设置了现金缓冲、税费准备金或运营储备目标，但不知道当前是否能按期达成，也不知道应该优先调整哪些支出。

## 4. v2 价值主张

FinCopilot v2 的价值不只是展示财务指标，而是帮助小微企业用户完成财务运营任务：

1. 整理交易和发票；
2. 分析现金流风险；
3. 识别异常支出；
4. 追问缺失信息；
5. 生成行动计划；
6. 跟踪处理进展；
7. 生成负责任的经营财务摘要。

## 5. 与普通 Chatbot 的区别

普通 chatbot 通常只回答用户问题。

FinCopilot v2 应体现 agentic workflow：

用户目标 → 任务规划 → 数据完整性检查 → 工具调用 → 缺失信息追问 → 行动计划生成 → 进展跟踪 → 结果摘要。

## 6. 安全边界

本系统仅用于财务整理、风险提醒和教育性支持，不构成投资、税务、法律、债务处置或专业财务建议。

系统不会：

- 认定任何交易为欺诈；
- 自动执行付款或转账；
- 提供投资建议；
- 提供税务建议；
- 提供法律建议；
- 承诺任何财务结果。
