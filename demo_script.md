# FinCopilot V2.2 Demo 演示脚本

## 1. 开场介绍

FinCopilot V2.2 是一个面向小微企业主的 Chat-Centric Multi-Agent 财务 Copilot。

它的目标是让用户不需要理解复杂财务系统，也能在一个主界面中完成：

1. 上传数据；
2. 提出问题；
3. 获取核心财务分析；
4. 查看风险和行动项；
5. 生成报告；
6. 必要时查看详细证据。

FinCopilot 不替代专业财务顾问，只用于财务整理、风险提醒和教育性支持。

## 2. 展示 Copilot 主界面

打开应用后，默认进入 Copilot 主界面。

说明 V2.2 的产品结构已经从多个功能页面并列，收敛为：

1. Copilot 主界面；
2. 分析详情；
3. 行动与报告；
4. 数据与设置。

主界面承担上传、提问、结果卡片、详细预览和后续追问。

## 3. 展示无数据状态与样例数据

在 Copilot 主界面展示：

- 数据状态；
- 新手引导；
- 上传提示；
- fallback 友好说明；
- 推荐任务卡片。

说明：如果用户不上传文件，系统可以使用默认样例数据体验完整流程。

## 4. 展示推荐任务

推荐任务会根据数据状态动态变化。

核心推荐任务包括：

- 检查未来 30 天现金流；
- 找出最可疑支出；
- 查看发票和付款优先级；
- 生成本周财务行动清单；
- 检查财务目标进度。

如果缺少数据，任务仍会显示，但会提示缺少哪些数据。

## 5. 运行第一次分析

在 Copilot 主界面点击推荐任务：

> 检查未来 30 天现金流

系统会自动执行问题：

> 未来 30 天现金流安全吗？

然后进入当前已有的 Multi-Agent 流程：

```text
Manager Agent
-> Tool Router
-> Specialist Agents
-> Response Composer
-> Safety Guardrails
-> Trace Builder
-> Action / Report 同步
```

## 6. 展示结果卡片

分析完成后，主界面展示：

- 结论 headline；
- 简短 summary；
- Agent 模式；
- 关键指标卡片；
- 主要风险；
- 建议行动；
- 仍需补充的信息；
- 详情入口；
- 报告入口。

说明：用户不需要阅读完整 JSON，也能理解本轮分析结论。

## 7. 展示详细结果预览

在同一页面展开“详细结果预览”，展示：

- 数据概览；
- 现金流与发票摘要；
- 异常支出摘要；
- 财务目标摘要；
- 行动项明细；
- Multi-Agent 报告预览；
- Agent 执行摘要；
- Tool Results 摘要。

说明：主界面已经足够完成第一轮理解，不必立刻跳转。

## 8. 展示分析详情页

进入“分析详情”。

展示五个 Tab：

1. 预算与分类；
2. 发票与现金流；
3. 异常支出；
4. 财务目标；
5. Agent 执行轨迹。

说明：分析详情页是完整证据层，适合查看表格、明细和 Agent 执行过程。

## 9. 展示行动与报告页

进入“行动与报告”。

展示：

- Chat-derived action items；
- 工作流行动项；
- 最近一次 Multi-Agent Report；
- 最近一次 Multi-Agent Trace；
- 报告和 Trace 下载入口。

说明：对话中的建议会沉淀为可跟踪行动项，而不是停留在聊天文本里。

## 10. 展示数据与设置页

进入“数据与设置”。

展示：

- 数据上传；
- 数据预览；
- 字段说明；
- Agent 与模型状态；
- 安全边界。

强调：Agent 与模型页只展示状态，不提供启用 / 关闭 Agent API 的用户开关。

## 11. 展示 fallback 机制

如果没有配置 `OPENAI_API_KEY`，系统不会中断。

页面会显示：

> 当前真实 Agent 暂不可用，系统将自动使用 fallback 分析。

fallback 模式仍然可以：

- 读取上传数据；
- 使用本地规则和工具摘要；
- 生成基础 Manager Plan；
- 生成 Specialist Agent fallback 输出；
- 展示结果卡片；
- 生成行动项和报告。

## 12. 总结 V2.2 创新点

FinCopilot V2.2 的核心变化：

1. 一个主界面完成主要任务；
2. 数据上传、推荐任务和自然语言提问整合；
3. Multi-Agent 输出卡片化；
4. 详细证据可在主界面展开；
5. 详情页变成完整表格、报告和设置归档；
6. Agent API 默认尝试启用，失败自动 fallback；
7. 新手引导降低使用门槛；
8. 安全边界贯穿所有页面。

## 13. 安全边界说明

FinCopilot V2.2 仅用于财务整理、风险提醒和教育性支持。

系统不会：

- 提供投资建议；
- 提供税务建议；
- 提供法律建议；
- 提供债务处置建议；
- 认定任何交易为欺诈；
- 承诺任何财务结果；
- 自动付款；
- 自动转账。

重要交易和财务决策请核查原始凭证，并在必要时咨询合格专业人士。

## 推荐演示问题

可以依次测试：

1. 未来 30 天现金流安全吗？
2. 这个月哪些支出最可疑？
3. 有哪些发票或付款要优先处理？
4. 帮我生成本周财务行动清单。
